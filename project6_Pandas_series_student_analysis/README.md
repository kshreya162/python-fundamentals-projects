# Pandas Series Student Marks Analysis

This project demonstrates how to use Pandas Series to analyze student marks.

## Concepts used

- Creating Series
- Handling missing values
- Statistical operations (mean, min, max, sum)
- Filtering data
- value_counts()
- unique() and nunique()
- apply() and lambda
- Categorizing data
- Sorting values
- Combining multiple Series

## File

student_marks_analysis.py

## Goal

Understand how Pandas Series can be used for basic data analysis.
