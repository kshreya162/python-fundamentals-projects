# Student Marks Analyzer (NumPy)

This project analyzes student marks using NumPy.

Concepts used:
- Arrays
- Indexing & slicing
- Mean, min, max
- Broadcasting
- Boolean filtering

Features:
- Calculates average marks of each student
- Finds highest and lowest marks
- Adds grace marks using broadcasting
- Identifies students scoring above threshold

Technologies:
Python, NumPy
